# Map Guessr

This is an iOS gaming app where players need to guess the name of a country within 5 attempts based on its map outline. With each attempt, the player will be given a clue on 
how far away the guessed country is from the actual country and in what direction.

## Game Features

- Single Play (Deployed)
- Play with Friends (Work in progress)
- Play Online (Backlog)
